## 💡 Lessons Learned

- Linux provides different commands for viewing user information (`id`, `who`, `w`, `last`), each serving a different purpose.
- `who` displays current login sessions, while `last` shows login history.
- System accounts generally have lower UIDs and are used by Linux services instead of human users.
- Linux uses the Unix Epoch as the reference point for storing time-related information.
- Chapter 16 shifts from understanding user accounts to creating and managing them.

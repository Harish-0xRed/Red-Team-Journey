### Domain Name System (DNS)

- DNS translates a **hostname** into an **IP address**.
- Linux stores DNS server information in:
  - `/etc/resolv.conf`
- `nameserver` specifies the DNS server IP.
- Multiple `nameserver` entries provide redundancy if one DNS server is unavailable.
- Used the `host` command to query DNS information.
- Observed that one domain (e.g., `google.com`) can return multiple IPv4 and IPv6 addresses due to **Load Balancing** and **High Availability**.

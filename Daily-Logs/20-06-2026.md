## Lessons Learned

* SSH login format:
  ssh username@hostname -p port

* "." represents the current directory

* "./filename" can be used to access files with special names

* Linux treats spaces as separators between arguments

* Quotes (" ") can be used to handle filenames containing spaces

* Backslash () can escape spaces in filenames

* "--" tells Linux to stop processing options and treat the following text as a filename

### Filesystem Hierarchy Standard (FHS)

Reviewed the purpose of important Linux directories.

🏠 `/` → Linux House

🚗 `/boot` → Boot Files

⚙️ `/etc` → Configuration Files

🔌 `/dev` → Hardware Devices

🧠 `/proc` → Running Process Information

📚 `/lib` → Shared Libraries

🏢 `/usr` → System Software

🏗 `/usr/local` → Third-Party Software

📦 `/opt` → Optional Applications

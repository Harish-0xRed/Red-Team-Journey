## 💡 Concepts Learned

### Linux Process Hierarchy
- Every running program is called a process.
- Every process has a unique Process ID (PID).
- Every process (except PID 1) has a Parent Process ID (PPID).
- Parent processes create child processes.

### PID & PPID
- PID = Unique identifier of a process.
- PPID = Parent Process ID.

### Parent & Child Processes
- systemd (PID 1) starts system services.
- login starts bash.
- bash starts user applications like Firefox.
- Processes form a family tree.

### Process Tree
- Understood how Linux organizes processes in a hierarchical tree.

## 🛠️ Commands Learned

```bash
pstree
```
Description:
Shows the Linux Process Family Tree.

```bash
pstree -p
```
Description:
Shows the Process Tree with Process IDs (PIDs).

```bash
ps
```
Description:
Displays a snapshot of running processes.

```bash
top
```
Description:
Displays running processes in real time.

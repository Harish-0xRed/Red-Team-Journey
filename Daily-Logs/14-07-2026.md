## 💡 Concepts Learned

### Memory Spaces

- Kernel Space → Protected memory for the Linux kernel.
- User Space → Memory where normal applications execute.
- Applications communicate with the kernel using System Calls.

Memory Tricks:

🔒 Kernel Space = Manager's Office

👨 User Space = Customer Hall

📞 System Call = Reception / Room Service

### Memory Commands

#### free

Description:
Displays a snapshot of RAM and Swap usage.

Memory Trick:
📸 Memory Photo

---

#### free -m

Description:
Displays memory usage in Megabytes (MB).

---

#### free -g

Description:
Displays memory usage in Gigabytes (GB).

---

#### free -s 10

Description:
Refreshes memory information every 10 seconds.

Memory Trick:
🎥 Repeated Memory Photos

---

### Swap

- Swap is disk space used when RAM becomes full.
- SSD is slower than RAM, so excessive swap usage reduces performance.

Memory Trick:

🏨 Extra Hotel Rooms

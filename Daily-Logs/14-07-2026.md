## 💡 Concepts Learned

### Linux Log Files

- Log files record important system events.
- Logging Daemon runs in the background and writes log events.
- Most log files are stored in `/var/log`.

Memory Tricks:

📒 Log File = Linux System Diary

👷 Daemon = Background Writer

📚 /var/log = Library of System Diaries

---

### Log Rotation

- Old log files are archived.
- Linux creates a new log file for future events.

Memory Trick:

📦 Old Diary → Archive

📖 New Diary → Continue Writing

---

### Kernel Messages

- Kernel generates hardware and boot-related messages.
- Kernel messages are temporarily stored in the **Kernel Ring Buffer (RAM)**.
- `dmesg` reads kernel messages directly from the Ring Buffer.
- Logging Daemon copies important events into `/var/log`.

Memory Trick:

🧠 Kernel Ring Buffer = Principal's Temporary Notebook

---

### Filesystem Hierarchy Standard (FHS)

- FHS defines how Linux organizes its directories.
- Every directory has a specific purpose.

Memory Tricks:

🏠 `/` = Entire Linux House

🚗 `/boot` = Car Key Box

⚙️ `/etc` = Settings Room

🔌 `/dev` = Hardware Room

🧠 `/proc` = Live Process Dashboard

🖥 `/sys` = Hardware Dashboard

📦 `/var` = Frequently Changing Files

🏠 `/home` = User Bedroom

## 💡 Lessons Learned
- A shell script is simply a collection of Linux commands stored in a text file.
- `sh script.sh` executes a script using the shell without requiring execute permission.
- `./script.sh` runs the script directly and requires execute permission.
- Linux error messages provide clues that help identify the actual problem.
- Effective troubleshooting starts by checking the file, its permissions, and its contents.

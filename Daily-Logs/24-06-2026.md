## Lessons Learned

* SSH login format:
  ssh username@hostname -p port

* "." represents the current directory

* ".." represents the parent directory

* Files beginning with "." are hidden

* ls -a displays hidden files

* Linux treats spaces as argument separators

* Quotes and backslashes can handle spaces in filenames

* "--" tells Linux to stop processing options and treat the following text as a filename

* The file command identifies file types

* ASCII text files are human-readable

## Today's Learning

### Linux Essentials - Chapter 10 (Working With Text)

#### Completed Topics

* cat command

  * View file contents
  * Understand STDIN behavior

* less command

  * Navigate large files
  * Search within files using `/`
  * Exit using `q`

* head and tail commands

  * Display first lines of a file
  * Display last lines of a file
  * Monitor files with `tail -f`

* Pipes (`|`)

  * Send output of one command to another
  * Practiced:

    * `ls /etc | head`
    * `ls /etc | tail`
    * `ls /etc | nl`

* Input / Output Redirection

  * `>` Redirect STDOUT
  * `>>` Append output
  * `2>` Redirect STDERR
  * `&>` Redirect both STDOUT and STDERR
  * `<` Redirect STDIN

* sort command

  * Alphabetical sorting
  * Numeric sorting with `-n`
  * Reverse sorting with `-r`
  * Field-based sorting using:

    * `-t` (delimiter)
    * `-k` (field)

* wc command

  * Count lines, words, and bytes
  * `wc -l`
  * `wc -w`
  * `wc -c`

* cut command

  * Extract fields from delimited files
  * `-d` delimiter option
  * `-f` field selection option
  * `-c` character position extraction

* grep command

  * Search matching lines
  * `grep`
  * `grep -c`
  * `grep -n`
  * `grep -v`
  * `grep -i`
  * `grep -w`

* Basic Regular Expressions (Introduction)

  * `.`
  * `[ ]`
  * `[^ ]`
  * `*`
  * `^`
  * `$`
